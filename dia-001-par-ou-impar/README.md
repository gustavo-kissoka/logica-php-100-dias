# Dia 001 Par ou ímpar + verificação de numero negativo ou positivo 

## Descrição:
- Programa que le um número e verificar se ele é *par* ou *impar*;
- Verifica se o número é *negativo* ou *positivo*;

Lógica:
- dividir o problema em 2 
- if/else if
- operador % (módulo)
- se N >= 0 (positivo) se não negativo
